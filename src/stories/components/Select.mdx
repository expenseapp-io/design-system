## Select With Arrows

In case you want to add an icon, you'll need to wrap the select component with `.select-container` class and after select, add `.select-icon` element with `<svg>` inside.

> Note:
> The SVG icon have a few requirenments:
>
> 1. The color of visible elements should be `currentColor`
> 2. The size of the icons should be `24px`. If the size is not equal to `24px` you might need to update paddings for `.select` to prevent showing icon on top of the selected text.

<Unstyled>
  <div className="select-container" style={{ width: 300 }}>
    <select className="select">
      <option value="no-icon">Select without arrows</option>
      <option value="today">Today</option>
      <option value="yesteday">Yesterday</option>
      <option value="thursday">Thursday</option>
    </select>
    <div className="select-icon">
      <CaretUpDownIcon />
    </div>
  </div>
</Unstyled>

```html
<div className="select-container">
  <select className="select">
    <option value="no-icon">Select without arrows</option>
```
