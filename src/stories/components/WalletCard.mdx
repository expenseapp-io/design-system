# Wallet Card

A `.wallet-card` component is made to show wallets details.

<UnstyledWithTheme>
  <div className="wallet-card">
    <div class="wallet-card-content">
      <div className="wallet-card-name">Chase</div>
      <div className="wallet-card-balance">1000 USD</div>
    </div>
  </div>
</UnstyledWithTheme>

You could use them as links to be able to navigate

<UnstyledWithTheme>
  <a href="#" className="wallet-card">
    <div class="wallet-card-content">
      <div className="wallet-card-name">Chase</div>
      <div className="wallet-card-balance">1000 USD</div>
    </div>
  </a>
</UnstyledWithTheme>
